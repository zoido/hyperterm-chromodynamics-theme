# hyperterm-chromodynamics-theme
Theme for Hyper.is based on https://github.com/MagicStack/Chromodynamics

## Preview

![preview](preview.png)

## Install

Add `'hyperterm-chromodynamics-theme'` to the plugins list in your `.hyper.js` config file.
